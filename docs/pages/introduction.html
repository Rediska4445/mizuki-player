Введение в Mizuka

Mizuka — это высокопроизводительный фреймворк на базе JavaFX, разработанный для создания современных десктопных приложений с премиальным пользовательским интерфейсом.

В отличие от стандартных подходов к разработке UI на JavaFX, Mizuka делает акцент на:

  • Сложных анимациях: плавные переходы, bounce-эффекты, параллельные переходы (ParallelTransition).
  • Кастомных визуальных эффектах: динамические тени, границы с градиентом, полупрозрачные оверлеи.
  • Строгом жизненном цикле: четкая система коллбэков onShow и onHide для интеграции бизнес-логики.
  • Темной теме: встроенные стили по умолчанию (цвет фона #1E1E1E).
Версия: 0.1.4.4 | Автор: Ebanina Std | Требования: Java 17+

Архитектура

Фреймворк построен по модульному принципу, напоминающему архитектуру Spring Framework. Основные компоненты:

  1. Root: Базовый класс корневой панели, управляющий глобальными интерполяторами (например, iceInterpolator).
  2. UI Elements: Набор готовых компонентов (в первую очередь AnimationDialog), инкапсулирующих сложную логику анимации.
  3. Events: Система событий для развязки отображения и логики (Event-Driven Architecture).

Пример инициализации

import rf.ebanina.UI.Root;
            import rf.ebanina.UI.UI.Element.AnimationDialog;
            import javafx.scene.paint.Color;

            public class MainApp extends Application {
            @Override
            public void start(Stage stage) {
                Pane root = new BorderPane();

                // Создание анимированного диалога
                AnimationDialog dialog = new AnimationDialog(stage, root);

                // Настройка визуального стиля
                dialog.setTopBorder(Color.web("#7092be"));

                // Подписка на жизненный цикл
                dialog.setOnShow(e -> System.out.println("Диалог полностью показан"));
                dialog.setOnHide(e -> System.out.println("Диалог закрыт, сохраняем данные"));

                // Показ с анимацией
                dialog.show();
            }
            }
