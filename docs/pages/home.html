Mizuka Framework

Mizuka — это высокопроизводительный фреймворк на базе JavaFX для создания десктопных приложений с премиальной анимацией, сложными визуальными эффектами и строгим управлением жизненным циклом UI.

Author: Ebanina Std
Version: 0.1.4.4
License: Proprietary

Ключевые особенности

  • Анимированные модальные диалоги с AnimationDialog
  • Кастомные интерполятор iceInterpolator для плавных переходов
  • Динамические тени и верхние рамки
  • Декларативная модель жизненного цикла (onShow, onHide)
  • Тёмная тема по умолчанию (#1E1E1E)

Быстрый пример

import rf.ebanina.UI.UI.Element.AnimationDialog;

AnimationDialog dialog = new AnimationDialog(stage, root);
dialog.setTopBorder(Color.web("#7092be"));
dialog.setOnShow(e -> loadData());
dialog.show();

Навигация по документации доступна в левой панели (кнопки и дерево).
