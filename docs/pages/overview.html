Mizuka Overview

Mizuka — это пет-проект: аудиоплеер, созданный на базе JavaFX, предназначенный для обработки аудиоданных и служащий средой для разработки сторонних сервисов (включая Mizuka Web).

О названиях: «Mizuka» и «Ebanina»

Эти имена не несут глубокого смыслового значения:

  • Mizuka — японское имя, заимствованное с одного из игровых серверов.
  • Ebanina — название, возникшее случайно и закрепившееся в проекте.

О документации

Данная документация описывает структуру, использование и возможности расширения приложения. Она будет регулярно обновляться по мере развития проекта.
