# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT create a public issue. 
Instead, send an email to: mizuka@mail.ru

I will try to acknowledge your report within 48 hours and keep you updated on the fix.
